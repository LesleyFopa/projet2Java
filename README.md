 Application de Gestion de Bibliothèque
 
  Application Java avec interface console pour gérer une bibliothèque complète avec base de données PostgreSQL.

  Fonctionnalités
  
-Gestion des Livres

-Ajouter, modifier, supprimer des livres

-Recherche par titre, auteur ou catégorie

-Gestion automatique des exemplaires disponibles

-Consultation du catalogue complet

  Gestion des Membres
  
-Inscription de nouveaux membres

-Recherche de membres

-Historique des emprunts

-Suppression de membres

  Gestion des Emprunts
  
-Enregistrement d'emprunts avec vérification de disponibilité

-Retour de livres avec calcul automatique de pénalités

-Affichage des emprunts en cours et en retard

-Système de pénalités pour retard

  Technologies Utilisées
  
-Java (JDK 22)

-PostgreSQL (Base de données)

-JDBC (Connexion base de données)

-Git (Contrôle de version)

  Prérequis
  
Logiciels requis :

-Java JDK 22 ou supérieur

-PostgreSQL 15 ou supérieur

-Git (pour cloner le projet)
